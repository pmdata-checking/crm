🏢 CRM Customer Importer

This bookmarklet runs on windowbase.info while you're logged in, lets you select OK customers, and imports them directly into your CRM's Supabase database.

How to install

  1. Make sure your browser bookmarks bar is visible (Ctrl+Shift+B)
  2. Drag the green button below onto your bookmarks bar
  3. It will appear as "Import to CRM"

Your Bookmarklet

Drag this to your bookmarks bar:

Import to CRM

↑ Drag the green button above to your bookmarks bar

How to use

  1. Log in to windowbase.info
  2. Go to Customers list page
  3. Change the "Show" dropdown to 200 so all customers are visible on one page
  4. Click "Import to CRM" in your bookmarks bar
  5. A panel will appear on the right showing all OK customers
  6. Tick the ones you want to import (or Select All)
  7. Click Import Selected
  8. Wait while it fetches each customer's details and users, then imports to Supabase
Note: Make sure to set Show to 200 before running the bookmarklet so all customers are visible. The importer reads the current page, then fetches address details and user information from each customer's detail page. Allow a few seconds per customer.
Column mapping: Company name → crm_customers, address fields → address columns, internal notes → notes. Each user → crm_contacts (split into first/last name) with last login stored in notes.
